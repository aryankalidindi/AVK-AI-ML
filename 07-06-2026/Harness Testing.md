# Weather App Benchmark

## Goal

Build a responsive weather app that demonstrates architecture, API integration, UI quality, and code organization.

### Requirements

- Next.js + TypeScript
    
- Tailwind CSS
    
- Search weather by city
    
- Browser geolocation
    
- Current weather
    
- 5–7 day forecast
    
- Hourly forecast
    
- Loading & error states
    
- Responsive UI
    
- Clean, reusable component structure
    

### Bonus

- AI-generated weather summary
    
- Save favorite cities
    
- Dark mode
    
- Unit tests
    

### Evaluation Criteria

- Project architecture
    
- Code quality
    
- UI/UX
    
- API integration
    
- Error handling
    
- Documentation
    
- Speed to completion
    
- Number of prompts/iterations required
    

**Deliverable:** A production-ready application with a clean folder structure, setup instructions, and a polished user experience.
